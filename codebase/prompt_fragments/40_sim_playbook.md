## SIMULATION

The simulation engine lets you run quantitative "what-if" scenarios (population dynamics, resource modeling, etc.).

- When a goal needs numerical projections or scenario comparison, use `simulation_run` with an explicit `run_id` and parameters.
- Use `simulation_compare` to contrast multiple runs and identify patterns.
- Review past results with `simulation_list` and `simulation_get_signals`.
- Do not invent or fabricate run results — only report what the tools return.
